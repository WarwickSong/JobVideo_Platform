## 1️⃣ 点赞逻辑（toggle）

```python
from sqlalchemy.orm import Session
from app.interactions.models import VideoLike
from app.video.models import Video


def toggle_video_like(
    db: Session,
    video_id: int,
    user_id: int
):
    """
    点赞 / 取消点赞

    功能：
    - 如果已点赞 → 删除记录（取消）
    - 如果未点赞 → 创建记录
    """

    like = db.query(VideoLike).filter(
        VideoLike.video_id == video_id,
        VideoLike.user_id == user_id
    ).first()

    if like:
        db.delete(like)
        db.commit()
        return False  # 当前状态：未点赞
    else:
        new_like = VideoLike(
            video_id=video_id,
            user_id=user_id
        )
        db.add(new_like)
        db.commit()
        return True  # 当前状态：已点赞
```

---

### 这段代码在系统中的意义

* 点赞是“状态行为”
* 不是布尔字段
* 而是一条存在与否的记录

👉 删除行 = 取消
👉 新建行 = 点赞

这叫做 **存在式状态设计**

from app.interactions.models import VideoView


def record_video_view(
    db: Session,
    video_id: int,
    user_id: int | None
):
    """
    浏览行为是事件，不去重。
    每播放一次就插入一条记录。
    """

    view = VideoView(
        video_id=video_id,
        user_id=user_id
    )
    db.add(view)
    db.commit()

## 1️⃣ 点赞接口

```python
@router.post("/{video_id}/like")
def like_video(
    video_id: int,
    db: Session = Depends(get_db),
    current_user: User = Depends(get_current_user)
):
    liked = toggle_video_like(
        db=db,
        video_id=video_id,
        user_id=current_user.id
    )

    like_count = db.query(VideoLike).filter(
        VideoLike.video_id == video_id
    ).count()

    return {
        "liked": liked,
        "like_count": like_count
    }
```

---

### 这一步做了什么？

1. 验证用户登录（必须）
2. 调用 service（核心逻辑）
3. 返回当前状态
4. 返回统计数（给前端直接渲染）

def view_video(
    video_id: int,
