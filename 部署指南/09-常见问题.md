# 常见问题

## 1. 域名打不开

常见原因：

- 域名还没生效
- DNS 解析错了
- 安全组没放行 80/443
- Nginx 没启动

## 2. 首页能打开，但视频加载失败

常见原因：

- `VIDEO_STORAGE_DIR` 配置错了
- Nginx `/videos/` 的 `alias` 配置错了
- 视频文件没有真正上传到服务器

## 3. 页面打开了，但视频流接口报错

常见原因：

- 后端服务没启动
- Nginx `/api/` 转发错了
- 后端 `.env` 没配置好

可以先检查：

```bash
sudo systemctl status jobvideo-backend
sudo journalctl -u jobvideo-backend -n 100 --no-pager
```

## 4. HTTPS 申请失败

常见原因：

- 域名还没解析到当前服务器
- 80 端口没通
- Nginx 配置错误

## 5. 修改了前端代码，但线上页面没变化

你可能只是改了源码，没有重新构建。

需要重新执行：

```bash
npm run build
cp -r dist/* /var/www/jobvideo/frontend-dist/
sudo systemctl reload nginx
```

## 6. 我完全不会判断哪里出错

你不用自己硬猜。

把下面这些信息发给我，我可以继续帮你定位：

- 你执行到哪一步
- 你执行的命令
- 完整报错信息
- `systemctl status` 输出
- `journalctl` 输出
- `nginx -t` 输出
