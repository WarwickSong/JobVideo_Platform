# 上线检查清单

上线前，请逐项确认。

## 基础访问

- [ ] 域名已经解析到服务器公网 IP
- [ ] 服务器安全组已开放 `80` 和 `443`
- [ ] Nginx 已启动
- [ ] 后端服务已启动

## 页面访问

- [ ] `http://你的域名` 能打开
- [ ] `https://你的域名` 能打开
- [ ] 首页视频流能加载
- [ ] 视频可以播放

## 接口访问

- [ ] `https://你的域名/api/` 路径可以转发到后端
- [ ] `https://你的域名/videos/...` 能访问到视频文件

## 业务验证

- [ ] 未登录用户可以浏览视频流
- [ ] 点赞/收藏未登录时会提示需要登录
- [ ] 后端生产环境已关闭 `test-token`

## 安全与配置

- [ ] `.env` 中的 `SECRET_KEY` 已修改
- [ ] `FRONTEND_ORIGINS` 已配置为你的真实域名
- [ ] 没有把开发环境配置直接暴露到公网

## 你可以接受的当前限制

现阶段还没有完整前端登录页，所以：

- [ ] 你接受第一版先以“可浏览、可公开访问”为目标
- [ ] 后续再补登录页面和更完整的账号功能
