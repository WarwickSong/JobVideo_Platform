# 购买服务器和域名

## 服务器怎么选

选择任意你熟悉的平台都可以，例如：

- 阿里云
- 腾讯云
- 华为云
- Vultr
- DigitalOcean

对新手最重要的不是品牌，而是：

- 能拿到公网 IP
- 能用 SSH 登录
- 能开放 `22`、`80`、`443` 端口
- 磁盘空间足够存放视频文件

## 推荐选择

建议：

- 系统：Ubuntu 24.04 LTS
- CPU：2 核或以上
- 内存：2GB 或以上
- 磁盘：40GB 或以上
- 地域：离你的主要访问者近一些
- 安全组开放：`22`、`80`、`443`

当前目标服务器信息：

- 域名：`zhenzhao.top`
- 服务器 IP：`124.220.216.17`

## 域名怎么选

买一个你自己能控制解析的域名。

买完后你至少需要能进入 DNS 管理界面。

## 域名解析要做什么

新增一条 `A` 记录：

- 主机记录：`@`
- 记录值：你的服务器公网 IP

如果你还想让 `www.你的域名` 也能访问，再加一条：

- 主机记录：`www`
- 记录值：你的服务器公网 IP

以当前目标域名为例：

- `zhenzhao.top` 指向 `124.220.216.17`
- `www.zhenzhao.top` 指向 `124.220.216.17`

## 做完后的结果

当你在浏览器输入你的域名时，请求会先到你的服务器。

这一步没完成，后面 HTTPS 和 Nginx 都没法正常验证。
