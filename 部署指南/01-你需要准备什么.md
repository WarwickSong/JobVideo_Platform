# 你需要准备什么

部署前，你需要准备以下东西。

## 必须有

1. 一台 Linux 云服务器
2. 一个域名
3. 能登录服务器的账号和密码，或者 SSH 密钥
4. 这个项目的完整代码
5. 你现有的视频文件目录

## 建议配置

第一版建议：

- 操作系统：Ubuntu 22.04 LTS
- CPU：2 核
- 内存：2GB 或以上
- 磁盘：40GB 或以上
- 带宽：按你预算选择

如果只是自己演示给别人看，这个配置够了。

## 你的本地电脑上最好也准备好

- Git
- Node.js
- Python 3.11 左右
- 一个能正常连接服务器的终端工具

Windows 常见工具：

- `PowerShell`
- `Windows Terminal`
- `PuTTY`
- `MobaXterm`

## 你现在不需要理解的东西

第一次部署时，不需要先搞懂这些概念：

- Docker
- Kubernetes
- CI/CD
- 对象存储
- 负载均衡
- 容器编排

先把单机版本上线，再逐步升级。
