# 服务器初始化

这一章是你第一次登录服务器后要做的事。

## 1. 登录服务器

如果你有公网 IP，常见命令是：

```bash
ssh root@你的服务器IP
ssh root@124.220.216.17
```

如果平台给你的不是 root 用户，就把 `root` 换成平台给你的用户名。

## 2. 更新系统

```bash
sudo apt update
sudo apt upgrade -y
```

## 3. 安装基础软件

当前项目需要 Node.js 22 LTS，因为前端使用 Vite 7。

```bash
sudo apt install -y curl wget git vim unzip lsof net-tools nginx python3 python3-venv python3-pip certbot python3-certbot-nginx
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs
node --version
npm --version
python3 --version
```

不需要安装 conda。服务器后端使用 Python 自带 `.venv` 即可。

## 4. 创建项目目录

```bash
sudo mkdir -p /var/www/zhenzhao.top
sudo chown -R $USER:$USER /var/www/zhenzhao.top
cd /var/www/zhenzhao.top
mkdir -p frontend-dist video_storage logs
```

最终建议目录结构：

```text
/var/www/zhenzhao.top/
  app/
    jobvideo_backend/
    jobvideo_frontend/
  frontend-dist/
  video_storage/
  logs/
```

## 5. 上传或拉取项目代码

方式 A：直接在服务器 `git clone`

```bash
cd /var/www/zhenzhao.top
git clone <你的 JobVideo_Platform 仓库地址> app
```

方式 B：本地压缩项目后上传到服务器，再解压到：

```text
/var/www/zhenzhao.top/app/
```

无论哪种方式，最终都要保证：

```text
/var/www/zhenzhao.top/app/jobvideo_backend
/var/www/zhenzhao.top/app/jobvideo_frontend
```

## 6. 准备视频文件目录

视频文件目录是：

```text
/var/www/zhenzhao.top/video_storage/
```

如果你已有视频文件，可以上传到这个目录。

后端 `.env` 中也要保持一致：

```env
VIDEO_STORAGE_DIR=/var/www/zhenzhao.top/video_storage
```

## 7. 启动 Nginx

```bash
sudo systemctl start nginx
sudo systemctl enable nginx
sudo systemctl status nginx
```

## 8. 这一章完成后，你应该具备这些条件

- 服务器能 SSH 登录
- 系统软件已经安装
- Node.js 版本满足 Vite 7 要求
- 项目代码已经在 `/var/www/zhenzhao.top/app`
- 视频目录已经准备好
- 域名已经解析到服务器
