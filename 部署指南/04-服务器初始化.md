# 服务器初始化

这一章是你第一次登录服务器后要做的事。

## 1. 登录服务器

如果你有公网 IP，常见命令是：

```bash
ssh root@你的服务器IP
```

如果平台给你的不是 root 用户，就把 `root` 换成平台给你的用户名。

## 2. 更新系统

```bash
sudo apt update
sudo apt upgrade -y
```

## 3. 安装基础软件

```bash
sudo apt install -y git nginx python3 python3-venv python3-pip nodejs npm certbot python3-certbot-nginx
```

## 4. 创建项目目录

```bash
sudo mkdir -p /var/www/jobvideo
sudo mkdir -p /var/www/jobvideo/video_storage
sudo chown -R $USER:$USER /var/www/jobvideo
```

## 5. 上传项目代码

有两种方式。

方式 A：直接在服务器 `git clone`

方式 B：你本地把项目压缩后上传

如果你暂时不会用 Git 仓库推送，方式 B 更直观。

## 6. 上传视频文件

把你本地的 `jobvideo_backend/video_storage/` 里的视频文件传到服务器：

```text
/var/www/jobvideo/video_storage/
```

## 7. 放置后端代码

建议放到：

```text
/var/www/jobvideo/backend/
```

## 8. 放置前端代码

建议放到：

```text
/var/www/jobvideo/frontend/
```

## 9. 这一章完成后，你应该具备这些条件

- 服务器能 SSH 登录
- 系统软件已经安装
- 项目代码已经上服务器
- 视频文件已经上传
- 域名已经解析到服务器
