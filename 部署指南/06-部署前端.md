# 部署前端

## 目标

把 Vue 前端构建成静态文件，交给 Nginx 提供。

## 1. 进入前端目录

```bash
cd /var/www/jobvideo/frontend/jobvideo_frontend
```

## 2. 安装依赖

```bash
npm install
```

## 3. 创建生产环境变量文件

参考仓库里的：

- [jobvideo_frontend/.env.production.example](/C:/Coding/JobVideo_Platform/jobvideo_frontend/.env.production.example)

服务器上创建：

```text
.env.production
```

内容：

```env
VITE_API_BASE_URL=/api
```

这样做的意思是：

- 浏览器访问你的前端页面时
- 前端请求会发到同域名下的 `/api`
- 再由 Nginx 转发给后端

## 4. 构建前端

```bash
npm run build
```

构建完成后会得到：

```text
dist/
```

## 5. 把构建结果放到统一目录

```bash
mkdir -p /var/www/jobvideo/frontend-dist
cp -r dist/* /var/www/jobvideo/frontend-dist/
```

## 6. 这一章完成后的状态

- 前端已经构建完成
- 静态文件已经准备好
- 接下来只差 Nginx 把它对外提供出来
