# 部署前端

## 目标

把 Vue 3 + Vite 前端构建成静态文件，交给 Nginx 提供。

当前项目使用 Vite 7，服务器需要 Node.js 20.19+ 或 22.12+，建议 Node.js 22 LTS。

## 1. 进入前端目录

```bash
cd /var/www/zhenzhao.top/app/jobvideo_frontend
```

## 2. 安装依赖

项目有 `package-lock.json`，生产构建建议使用：

```bash
npm ci
```

如果 `npm ci` 因锁文件问题失败，再改用：

```bash
npm install
```

不需要安装 Vue CLI，因为当前项目不是 Vue CLI 项目。

## 3. 创建生产环境变量文件

服务器上创建：

```text
/var/www/zhenzhao.top/app/jobvideo_frontend/.env.production
```

内容：

```env
VITE_API_BASE_URL=/api
```

这样做的意思是：

- 浏览器访问前端页面时
- 前端请求会发到同域名下的 `/api`
- Nginx 再把 `/api/video/feed` 转发为后端真实接口 `/video/feed`

## 4. 构建前端

```bash
npm run build
```

构建完成后会得到：

```text
dist/
```

## 5. 把构建结果放到统一目录

```bash
rm -rf /var/www/zhenzhao.top/frontend-dist/*
cp -r dist/* /var/www/zhenzhao.top/frontend-dist/
```

## 6. 这一章完成后的状态

- 前端已经构建完成
- 静态文件已经放到 `/var/www/zhenzhao.top/frontend-dist`
- 接下来由 Nginx 把它对外提供出来
