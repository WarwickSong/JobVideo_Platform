# 部署后端

## 目标

让 FastAPI 后端在服务器后台稳定运行。

当前项目后端目录是：

```text
/var/www/zhenzhao.top/app/jobvideo_backend
```

## 1. 进入后端目录

```bash
cd /var/www/zhenzhao.top/app/jobvideo_backend
```

## 2. 创建虚拟环境

```bash
python3 -m venv .venv
source .venv/bin/activate
```

服务器不需要安装 conda，使用 Python 自带 `.venv` 即可。

## 3. 安装依赖

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

## 4. 创建 `.env`

在服务器里新建：

```text
/var/www/zhenzhao.top/app/jobvideo_backend/.env
```

内容：

```env
APP_ENV=production
SECRET_KEY=换成你自己的长随机字符串
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=60
DATABASE_URL=sqlite:///./test.db
VIDEO_STORAGE_DIR=/var/www/zhenzhao.top/video_storage
FRONTEND_ORIGINS=https://zhenzhao.top,https://www.zhenzhao.top
ENABLE_TEST_TOKEN=false
```

生成随机密钥：

```bash
openssl rand -hex 32
```

## 5. 准备视频目录

```bash
sudo mkdir -p /var/www/zhenzhao.top/video_storage
sudo chown -R $USER:$USER /var/www/zhenzhao.top/video_storage
sudo chmod -R 755 /var/www/zhenzhao.top/video_storage
```

## 6. 手动测试启动

```bash
cd /var/www/zhenzhao.top/app/jobvideo_backend
source .venv/bin/activate
uvicorn app.main:app --host 127.0.0.1 --port 8000
```

新开一个终端窗口，执行：

```bash
curl http://127.0.0.1:8000/
curl http://127.0.0.1:8000/video/feed
```

如果返回 JSON，说明后端基本启动成功。测试完成后按 `Ctrl+C` 停止手动启动的服务。

## 7. 可选：创建演示账号

当前前端有演示登录面板。如果你希望线上演示账号可用，可以执行：

```bash
cd /var/www/zhenzhao.top/app/jobvideo_backend
source .venv/bin/activate
python create_demo_users.py
```

如果这是正式生产站点，不建议长期公开演示账号密码。

## 8. 创建 systemd 服务

创建文件：

```bash
sudo vim /etc/systemd/system/zhenzhao-backend.service
```

内容如下。如果服务器用户名不是 `ubuntu`，把 `User=ubuntu`、`Group=ubuntu` 改成你的实际用户名。

```ini
[Unit]
Description=JobVideo FastAPI Backend
After=network.target

[Service]
Type=simple
User=ubuntu
Group=ubuntu
WorkingDirectory=/var/www/zhenzhao.top/app/jobvideo_backend
EnvironmentFile=/var/www/zhenzhao.top/app/jobvideo_backend/.env
ExecStart=/var/www/zhenzhao.top/app/jobvideo_backend/.venv/bin/uvicorn app.main:app --host 127.0.0.1 --port 8000
Restart=always
RestartSec=3

[Install]
WantedBy=multi-user.target
```

## 9. 启动服务

```bash
sudo systemctl daemon-reload
sudo systemctl enable zhenzhao-backend
sudo systemctl start zhenzhao-backend
sudo systemctl status zhenzhao-backend
```

## 10. 如果启动失败

查看日志：

```bash
sudo journalctl -u zhenzhao-backend -n 100 --no-pager
```

重点检查：

- `.env` 是否存在
- `.venv` 是否创建成功
- `WorkingDirectory` 是否正确
- `ExecStart` 是否指向 `.venv/bin/uvicorn`
- `VIDEO_STORAGE_DIR` 是否存在且有权限
